# kadai07_basic_php

## ②課題内容（どんな作品か）
- 現場向けアンケートアプリとして作成しました。
熱中症対策アンケートでは、作業場所ごとに必要な対策グッズを集計できます。
ヒヤリハット報告では、現場で発生した危険事例を記録・可視化できます。
PHPを用いて入力→保存→表示の基本的な流れを実装しました。

## ⑤工夫した点・こだわった点
- 今回は自分の業務経験を活かして、建設現場向けのアンケートアプリを作成しました。
熱中症対策とヒヤリハット報告という、現場でリアルに起こる課題をテーマにしています。
特に熱中症対策アンケートでは、作業場所ごとのニーズが見えるように投票結果を集計表示できるよう工夫しました。
PHPの基本である「入力→保存→表示」の流れを理解しながら、少し実用性も意識して作成できたと思います。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 今回の課題で最も難しかった点は、PHPの処理の流れを理解することでした。
特に、入力画面・保存処理・表示画面をそれぞれ別ファイルで管理しながら、データがどのように受け渡されているかを理解するのに苦戦しました。
